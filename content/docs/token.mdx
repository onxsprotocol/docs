## ONXS Token

The ONXS token powers the Oorth Nexus ecosystem. It is used for transaction fees, staking rewards, and governance voting.

## Token Facts

- Token symbol: ONXS
- Total supply: 1,000,000,000
- Staking APY: up to 12%
- Governance: DAO voting

## Utility

- Pay for protocol transactions and network services.
- Stake ONXS to secure the network and earn rewards.
- Participate in governance decisions that shape the protocol.
